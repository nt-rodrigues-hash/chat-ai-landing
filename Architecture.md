# Application Architecture Best Practices

## Layered Architecture
- Separate concerns: presentation, business, data layers.
- Use contracts (interfaces) between layers.

## Microservices vs Monolith
- Choose microservices for scalability and isolation.
- Monolith for smaller teams, simpler deployment.

## Design Patterns
- Repository, Service, Factory.
- Singleton should be avoided where possible.
