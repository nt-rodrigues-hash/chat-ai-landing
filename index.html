Sobre o Projeto

CSS Social Media

Um projeto de estudo e compartilhamento de conhecimento sobre CSS avançado.

Tecnologias utilizadas:

  • HTML5 semântico
  • CSS Moderno (Flexbox, Grid, Custom Properties)
  • JavaScript Vanilla (ES6+)
  • LocalStorage para persistência

Recursos:

  • Chat: Conversas interativas com demonstrações de CSS
  • Documentação: Notas técnicas organizadas
  • Exemplos: Demonstrações práticas e interativas
  • Design responsivo com tema escuro
